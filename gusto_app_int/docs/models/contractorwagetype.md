# ContractorWageType

The contractor's wage type, either "Fixed" or "Hourly".

## Example Usage

```python
from gusto_app_integration.models import ContractorWageType

value = ContractorWageType.FIXED
```


## Values

| Name     | Value    |
| -------- | -------- |
| `FIXED`  | Fixed    |
| `HOURLY` | Hourly   |
