## get

Get an employee's pay stubs

scope: `pay_stubs:read`

### Example Usage

```python
from gusto_embedded import Gusto
import os

with Gusto(
    company_access_auth=os.getenv("GUSTO_COMPANY_ACCESS_AUTH", ""),
) as gusto:

    res = gusto.pay_stubs.get(employee_id="<id>")

    # Handle response
    print(res)

```
