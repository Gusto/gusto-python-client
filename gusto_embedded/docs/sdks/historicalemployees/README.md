# HistoricalEmployees
(*historical_employees*)

## Overview

### Available Operations

* [update](#update) - Update a historical employee

## update

Update a historical employee, an employee that was previously dismissed from the company in the current year.

scope: `employees:manage`

### Example Usage

```python
from datetime import date
from gusto_embedded import Gusto
import os


with Gusto(
    company_access_auth=os.getenv("GUSTO_COMPANY_ACCESS_AUTH", ""),
) as gusto:

    res = gusto.historical_employees.update(company_uuid="<id>", historical_employee_uuid="<id>", version="db0edd04aaac4506f7edab03ac855d56", first_name="Soren", last_name="Kierkegaard", date_of_birth="1995-05-05T00:00:00Z", ssn="123456294", work_address={
        "location_uuid": "1da85d35-1910-4d5e-8e94-39e8cdfe8c9a",
    }, home_address={
        "street_1": "300 3rd Street",
        "street_2": None,
        "city": "San Francisco",
        "state": "CA",
        "zip": "94107",
    }, termination={
        "effective_date": date.fromisoformat("2022-09-15T00:00:00Z"),
    }, job={
        "hire_date": date.fromisoformat("2018-05-09T00:00:00Z"),
    }, middle_initial="A", employee_state_taxes={
        "wc_covered": True,
        "wc_class_code": "20992",
    })

    # Handle response
    print(res)

```
