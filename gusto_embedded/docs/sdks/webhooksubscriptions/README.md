## delete

>

scope: `webhook_subscriptions:write`


### Example Usage

```python

from gusto import Gusto
import os

with Gusto() as g_client:

    g_client.webhook_subscriptions.delete(security=gusto.DeleteV1WebhookSubscriptionUUIDSecurity(
```
