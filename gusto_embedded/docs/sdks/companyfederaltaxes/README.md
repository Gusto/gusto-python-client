## get_details

Fetches attributes relevant for a company's federal taxes.

scope: `company_federal_taxes:read`

### Example Usage

```python
from gusto_embedded import Gusto
import os

with Gusto(
    company_access_auth=os.getenv("GUSTO_COMPANY_ACCESS_AUTH", ""),
```
