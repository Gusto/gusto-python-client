# CompanyAttachment

## Overview

### Available Operations

* [get_download_url](#get_download_url) - Get a temporary url to download the Company Attachment file

## get_download_url

Retrieve a temporary url to download an attachment file uploaded by the company.

### Related guides
- [Manage company attachments](doc:manage-company-attachments)

scope: `company_attachments:read`

### Example Usage

<!-- UsageSnippet language="python" operationID="get-v1-companies-attachment-url" method="get" path="/v1/companies/{company_id}/attachments/{company_attachment_uuid}/download_url" -->
```python
import gusto_embedded
from gusto_embedded import Gusto
import os


with Gusto(
    company_access_auth=os.getenv("GUSTO_COMPANY_ACCESS_AUTH", ""),
) as gusto:

    res = gusto.company_attachment.get_download_url(company_id="<id>", company_attachment_uuid="<id>", x_gusto_api_version=gusto_embedded.GetV1CompaniesAttachmentURLHeaderXGustoAPIVersion.TWO_THOUSAND_AND_TWENTY_FIVE_MINUS_06_MINUS_15)

    # Handle response
    print(res)

```
