# CompanyAttachmentSDK
(*company_attachment*)

## Overview

### Available Operations

* [get_download_url](#get_download_url) - Get a temporary url to download the Company Attachment file

## get_download_url

Retrieve a temporary url to download a attachment file uploaded
by the company.

scope: `company_attachments:read`

### Example Usage

```python
from gusto import Gusto
import os

with Gusto(
    company_access_auth=os.getenv("GUSTO_COMPANY_ACCESS_AUTH", ""),
) as g_client:

    res = g_client.company_attachment.get_download_url(company_id="<id>", company_attachment_uuid="<id>")

    # Handle response
    print(res)

```
